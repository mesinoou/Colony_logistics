Colony Logistics style pack

This is the active MineColonies/Structurize style pack bundled with the Colony Logistics mod.
It replaces the old development pack `colony_logistics_dev`. The runtime installer deletes the legacy folder from the active instance and installs this pack under `blueprints/colony_logistics`.

Blueprint files are provided in both locations for Structurize/MineColonies compatibility:
- colony_logistics/logistics_office1.blueprint ... trade_terminal5.blueprint
- colony_logistics/huts/logistics_office1.blueprint ... trade_terminal5.blueprint
